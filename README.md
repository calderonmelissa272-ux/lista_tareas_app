# Lista de Tareas - Tkinter

Aplicación de escritorio desarrollada en Python con Tkinter para gestionar tareas diarias.

## Funcionalidades

- Agregar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Visualización de tareas pendientes y completadas

## Atajos de teclado

- Enter → Agregar tarea
- C → Marcar como completada
- Delete → Eliminar tarea
- Escape → Cerrar aplicación

## Arquitectura

El proyecto sigue una arquitectura modular por capas:

- modelos → Representación de datos
- servicios → Lógica de negocio
- ui → Interfaz gráfica
- main → Punto de entrada

python main.py
